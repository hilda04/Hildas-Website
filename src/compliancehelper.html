Use Case

Instead of yearly screenshot hunts for firewall rules, encryption settings, or IAM policies, Compliance Helper continuously captures them, signs artifacts, and maps them to ISO 27001 Annex A and PCI DSS v4.0 controls. When audit time comes, export the mapped pack and share with your auditor.

Evidence Types

  • Configurations (e.g., S3 bucket policies, KMS key rotation, SG/NACL rules)
  • Logs (e.g., CloudTrail events, access logs, change history)
  • Screenshots (e.g., console settings with timestamp & user)
  • Documents (e.g., policies, SOPs, risk assessments)

Data Sources (examples)

  • AWS: Config, CloudTrail, Security Hub, IAM, S3, KMS
  • Azure: Defender for Cloud, Activity Logs, Key Vault
  • G Suite / O365: Admin audit logs
  • On‑prem: Syslog, AD, firewall exports

*Metrics are indicative; actual improvements vary by environment and process maturity.
